1.iframe和a标签以及target属性
    iframe是支持相对路径.
    例1：<iframe name = "xxx" src = "" frameborder = "0"></iframe>
             <a herf = "" target = "xxx"></a>点击a标签的时候后会在iframe里出现一个新页面。因为用了name属性。另外frameborder = “0”是为了消除iframe自带的一个边框。
    例2： 创建一个index.html,在html的body中嵌入一个<iframe src = "./index2.html"></iframe>然后在index2.html里面嵌入一个<iframe src = "./index3.html"></iframe>
              最后在index3.html里面写上1、<a href = "http://qq.com" target =_blank></a>
                                                          2、<a href = "http://qq.com" target =_self></a>
                                                          3、<a href = "http://qq.com" target =_parent></a>
                                                          4、<a href = "http://qq.com" target =_top></a>
               这四个标签会分别在 “新窗口”、“index3.html窗口”、“index2.html窗口”、“当前窗口”打开。
2.a标签的href属性
    <a href="//qq.com">QQ</a>这里的a标签地址是“无协议地址”，当你的浏览器目前用的是什么协议就会自动继承协议，http就继承http,file就会技能file协议。
    我们可以载终端上输入  npm i -g http-server这样可以安装一个服务器，与之前用node.js写的server是一样的。
    href的值:1、<a href="?name=xxx">a</a>点击后会在当前的页面发起?name=xxx的请求
                  1、<a href="#xxx">a</a>点击后不会发起请求，但是会加入锚点，因为锚点只在当前页面跳转。
                  1、<a href="/..">a</a>点击后会发起GET /HTTP/1.1的请求，路径是/，当路径是根目录就无法往前退了。
    伪协议<a href="javascript:;"></a>点击之后不会有任何变化，（满足奇怪的需求）。另外javascript里可以加入js代码，点击后会自动触发js代码的动作。
3.form标签
        form标签有一个method属性设置"POST"的值可以发送post请求。但是发送请求必须在form标签内有一个<input type="submit">才能提交。同理button标签也能提交，但是不能加入其他属性，否则会出问题。
例：<form action="users" method="post">
        <input type="text"name="username" value="username">
        <input type="password" name="password" value="passwrod">
        <Input type="submit" value="提交"
    
        </form>
         name属性会被带到第四部分显示。
        
        label属性会提升交互的友好度。公认的写法：<label><input type="text">输入</label>
        例：复选框：<label><input type="checkbox" name="fruit" value="banana">香蕉</label>
                            <label><input type="checkbox" name="fruit" value="apple">苹果</label>
               

                   单选：     <label><input type="radio" name="sex" value="man">男</label>
                                <label><input type="radio" name="sex" value="woman">女</label>
                    
4.select标签
    例：<select name="group" multiple>
<option value=""></option>
<option value="1">第一组</option>
<option value="2">第二组</option>
<option value="3" disabled>第三组</option>
<option value="4" selected>第四组</option>
</select>
multiple代表可以同时多选,disabled代表不能被选中，selected代表默认首选。

6.table标签
例： <table border="1">
  <thead>
    <tr>
      <th>序号</th><th>姓名</th><th>年龄</th><th>性别</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th></th><td>宋萌</td><td>23</td><td>男</td>
    </tr>
    <tr>
      <th></th><td>张三</td><td>23</td><td>男</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <th>总人数</th><td></td><td></td><td>2</td>
    </tr>
    <tr>
      <th>平均年龄</th><td></td><td></td><td>23</td>
    </tr>
  </tfoot>    
  </table>
HTML标准规定table标签里只能有thead、tbody、tfoot以及一个colgroup标签，colgroup不好用。
tr全称“table row 代表独占一行”td全称“table data代表数据”。
